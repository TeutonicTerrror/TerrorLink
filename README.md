# TerrorLink

Private, ephemeral group chat platform. Host a chat server through a Cloudflare Tunnel and invite others via a shareable link. Messages are temporary and not persisted.

Built with Electron, Express, and WebSockets.

## Project Structure

```
scripts/                 Build, dev, and cleanup scripts (run from project root)
terrorlink-client/       Electron desktop application
  src/main/              Main-process code (window management, IPC, server orchestration)
  src/renderer/          Renderer-process code (UI, chat logic, styles)
  assets/                Icons, backdrops, sounds, and cloudflared executable
  vendor/                Third-party credits and assets
  build/                 Electron-builder output and installer icon

terrorlink-server/       Chat server (Express + WebSocket)
  src/                   Server source code
  server.ncc.js          Bundled server (generated by build step)

TLHook/                  Native Windows keyboard hook addon
  src/                   C++ source and Node.js binding
  build/                 Compiled native module (generated by node-gyp)
```

## Requirements

- Node.js 22+
- npm
- Windows (required for TLHook native addon and NSIS installer)
- PowerShell 5+

## Setup

Install dependencies for all three components:

```
powershell -ExecutionPolicy Bypass -File scripts/setup.ps1
```

## Build the Server Bundle

Bundle the server with `@vercel/ncc` into a single file:

```
powershell -ExecutionPolicy Bypass -File scripts/build-server.ps1
```

## Run in Development Mode

Run the Electron app directly without building an installer. The server runs as a child process spawned by the client.

```
powershell -ExecutionPolicy Bypass -File scripts/dev.ps1
```

After closing the app, run the cleanup script to remove temporary server copies:

```
powershell -ExecutionPolicy Bypass -File scripts/cleanup.ps1
```

## Build the Installer

Produces a Windows NSIS installer executable and portable version.

```
powershell -ExecutionPolicy Bypass -File scripts/build-installer.ps1
```

The installer EXE and `latest.yml` update manifest will be in `terrorlink-client\dist\`. Run the cleanup script afterwards to remove temp build files:

```
powershell -ExecutionPolicy Bypass -File scripts/cleanup.ps1
```

## Build TLHook (Native Addon)

Compile the keyboard hook native module from C++ source:

```
powershell -ExecutionPolicy Bypass -File scripts/build-tlhook.ps1
```

Note: The TLHook addon is compiled automatically during setup. Rebuild if you modify `TLHook.cc` or update Node.js.

## Cleanup

Removes all build artifacts, temp copies, and leftover files:

```
powershell -ExecutionPolicy Bypass -File scripts/cleanup.ps1
```

This removes:
- `terrorlink-client/dist/` (installer output)
- Temporary server copies inside `terrorlink-client/terrorlink-server/`
- Server node_modules backup left by the build step
- TLHook build intermediates (preserves `TLHook.node`)
- Generated `tailwind.css` and debug logs

To also remove all `node_modules/` folders (full reset):

```
powershell -ExecutionPolicy Bypass -File scripts/cleanup.ps1 -CleanDeps
```
